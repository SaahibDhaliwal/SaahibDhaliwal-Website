Project 2: E - Compare

I developed a web application that displays relevant product details from top e-commerce websites like Amazon and Best Buy so you can compare them! I utilized the Selenium and BeautifulSoup Python libraries to make web scrapers that would interact with websites and extract the relevant details. I made the UI for the application by using HTML, CSS, and Bootstrap and I used Flask for the backend server. I also had to implement the Jinja2 templating engine so that product details, that were stored and manipulated using Pandas, could be dynamically rendered in tables
